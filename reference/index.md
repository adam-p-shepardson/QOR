# Package index

## QOR Functions

Each Step of the QOR Method has its own function:

- [`query()`](https://adam-p-shepardson.github.io/QOR/reference/query.md)
  : Geocode Unit (Voter) Addresses to Longitude/Latitude Coordinates
- [`overlay()`](https://adam-p-shepardson.github.io/QOR/reference/overlay.md)
  : Match Point Geometries to Polygon Geometries (e.g., Voter Locations
  to School Districts)
- [`recover()`](https://adam-p-shepardson.github.io/QOR/reference/recover.md)
  : Recover Failed Point-to-Polygon Matches Using Zipcodes
